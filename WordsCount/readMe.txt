Program is written in Java.

1. It scans all files in "resource" directory.
2. Iterates through all files.
3. Collect all words from each file.
4. Calculates amount repeatings of each word.
5. Sortes and groups by first letter (intervals for words groupping are described in "BusinessConstants.java" class, "LETTERS_FOR_DIVIDE" constant).
6. Saves all groupped words and appropriate amounts in appropriate files.
